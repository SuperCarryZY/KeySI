# KeySI UI 改进说明

## 最新更新 (v2.1)

### 🎯 BertTopic降维支持
- **新增功能**: 集成BertTopic进行关键词降维，提供更智能的主题建模
- **自动回退**: 如果BertTopic不可用，自动回退到t-SNE方法
- **安装说明**: 运行 `pip install -r requirements_berttopic.txt` 安装BertTopic依赖

### 🔧 界面简化
- **移除上下移动按钮**: 简化关键词组管理，只保留删除(×)按钮
- **保持添加逻辑**: 点击2D图表中的关键词仍然会添加到选中组，而不是替换

## 主要功能特性

### 1. 关键词2D可视化
- **左侧60%**: 交互式2D关键词降维图表
- **右侧40%**: 分组管理界面
- **悬停效果**: 鼠标悬停显示具体关键词
- **点击选择**: 点击图表中的点将关键词添加到选中组

### 2. 智能降维
- **BertTopic优先**: 使用主题建模进行语义降维
- **t-SNE备选**: 如果BertTopic失败，自动使用t-SNE
- **动态调整**: 根据关键词数量自动调整降维参数

### 3. 分组管理
- **动态创建**: 可指定组数量并自动生成
- **关键词添加**: 支持从2D图表点击添加和手动输入添加
- **删除功能**: 每个关键词都有删除(×)按钮
- **组选择**: 点击组标题选择当前操作组

### 4. 高级功能
- **GPU加速**: 支持CUDA和MPS设备
- **批处理**: 优化大规模数据处理
- **内存管理**: 自动GPU内存清理
- **模型训练**: 支持BERT模型微调

## 技术架构

### 降维流程
1. **关键词提取**: 使用KeyBERT从文档中提取关键词
2. **嵌入计算**: 使用SentenceTransformer生成关键词嵌入
3. **智能降维**: 优先使用BertTopic，失败时回退到t-SNE
4. **可视化**: 生成2D散点图供用户交互

### 交互逻辑
1. **点击选择**: 用户在2D图表中点击关键词
2. **组分配**: 关键词自动添加到当前选中的组
3. **数据同步**: 更新`group-data`和`group-order`存储
4. **界面刷新**: 自动更新分组显示和图表状态

## 使用说明

### 基本操作流程
1. **启动应用**: 运行`finaluimodified.py`
2. **设置组数**: 在"Enter number of groups"中输入所需组数
3. **生成组**: 点击"Generate Groups"创建空组
4. **选择组**: 点击组标题选择要操作的组
5. **添加关键词**: 
   - 在2D图表中点击关键词点
   - 或在输入框中手动输入关键词
6. **删除关键词**: 点击关键词旁的"×"按钮

### 高级功能
- **GPU优化**: 自动检测并使用可用的GPU设备
- **批处理**: 支持大规模文档处理
- **模型训练**: 可训练自定义BERT模型

## 故障排除

### 常见问题
1. **BertTopic导入失败**
   - 运行: `pip install bertopic`
   - 检查Python版本兼容性

2. **GPU内存不足**
   - 系统会自动回退到CPU模式
   - 或调整批处理大小

3. **界面显示异常**
   - 检查浏览器兼容性
   - 确保所有依赖正确安装

### 性能优化
- **GPU使用**: 确保CUDA或MPS正确安装
- **内存管理**: 大文件处理时注意内存使用
- **批处理**: 根据设备性能调整批处理大小
